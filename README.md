# itholics/oxid-debug-smarty

Helper to debug smarty templates in OXID


## Installation

```bash
composer require --dev -n itholics/oxid-debug-smarty
```

## Usage
Add following function tag at the place you want xdebug to add a break point.
```smarty
[{xdebug_break}]
```
